---
title: Work Locally
description: "Run your Modus app locally and access Hypermode models"
---

When you are building Modus apps locally, you can use the Hyp cli to access
Hypermode-hosted models defined in your [app's manifest](/modus/app-manifest).

## Hyp CLI setup

<Steps>
   <Step title="Install Hyp CLI">
   First, install the Hyp CLI to augment Modus:

   <CodeGroup>

```bash MacOS + Linux
curl -sSL http://install.hypermode.com/hyp.sh | bash
```

```js Windows
npm install -g @hypermode/hyp-cli
```

   </CodeGroup>
   </Step>

<Step title="Log in to the Hyp CLI">
  Ensure you're logged in to the Hyp CLI with the `hyp login` command.
</Step>

   <Step title="Run your app locally">
   Run `modus dev` to run your Modus app locally.
   </Step>
</Steps>
