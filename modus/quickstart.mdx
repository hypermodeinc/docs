In this quickstart we'll show you how to:

- Get setup with Modus and its CLI.
- Use the most basic and common components of Modus: manifest, functions, and API calls.
- Build a simple application with Modus.
- Deploy your application to the cloud using Hypermode.

## Prerequisites

- Node.js - v22 or higher.
- Text editor - We recommend [VS Code](https://code.visualstudio.com/).
- Terminal - Modus is accessed through its command-line interface (CLI).

## CLI Installation

<CodeGroup>

```bash cURL
curl -sSL http://install.hypermode.com/modus.sh | bash
```

```js npm
npm install -g @hypermode/modus
```

</CodeGroup>
